## Getting Started

First, run the development server:
# techstack:
- nextJs 16.0.3
- Zustand 
- React-Query 
- clerk (authentication)
- resed (invitation by mail)
- postgress, prisma
- middleware

# Install pacakge and run project
```bash

npm i

```

```bash

npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
